# 🏭 Manufacturing Performance Dashboard – Power BI

This repository contains a comprehensive Power BI dashboard designed to monitor and analyze manufacturing operations through interactive visualizations and KPIs. The dashboard provides insights into production performance, quality management, machine utilization, and operational efficiency, enabling data-driven decision-making and continuous process improvement.

### 🔍 Dashboard Overview

The dashboard includes the following key components:

- **Production Performance Analysis**
  - Total Production Monitoring
  - Monthly Production Trends
  - Shift-wise Production Distribution
  - Product and Machine Performance Analysis

- **Quality Management**
  - Defect Rate Tracking
  - Defect Analysis by Machine and Shift
  - Operator-wise Quality Performance
  - Production vs Defect Comparison

- **Machine Utilization**
  - Average Efficiency Monitoring
  - Downtime Analysis
  - Machine-wise Efficiency Comparison
  - Downtime Distribution by Shift

- **Detailed Operational Insights**
  - Production Quantity
  - Defect Percentage
  - Downtime Hours
  - Efficiency Metrics
  - Machine and Operator-Level Data

---

### 💡 Use Cases

- Monitor manufacturing productivity and efficiency
- Identify defect trends and quality issues
- Analyze machine performance and downtime
- Evaluate shift and operator effectiveness
- Support operational and management decision-making
